# projetoAgrinho_Luis_1-B
🌱 EcoLar é um site de imobiliária sustentável desenvolvido com HTML, CSS e JavaScript. O projeto apresenta imóveis ecológicos, informações sobre sustentabilidade e recursos de acessibilidade, como leitor de tela, ajuste de fonte, modo escuro e alto contraste, promovendo inclusão e conscientização ambiental.
